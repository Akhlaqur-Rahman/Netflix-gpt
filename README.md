







   # NetflixGPT 🎬🤖

NetflixGPT is a Netflix-inspired movie browsing web application powered by GPT-based search.
It allows users to explore trending movies, watch trailers, and get intelligent movie recommendations.

The application integrates TMDB APIs for movie data and provides a Netflix-like UI experience.

## ✨ Features

* 🔐 User Authentication (Sign In / Sign Up)
* 🎬 Browse Trending Movies
* 🎥 Watch Movie Trailers
* 🤖 GPT Powered Movie Search
* ❤️ Add Movies to My List
* 📱 Fully Responsive UI
* 🎨 Netflix-style Interface

---

## 🛠 Tech Stack

Frontend

* React.js
* Redux Toolkit
* Tailwind CSS

APIs & Services

* TMDB API
* OpenAI GPT API
* Firebase Authentication

Deployment

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/Akhlaqur-Rahman/Netflix-gpt.git
```

Go to project folder

```
cd Netflix-gpt
```

Install dependencies

```
npm install
```

Start development server

```
npm run dev
```
